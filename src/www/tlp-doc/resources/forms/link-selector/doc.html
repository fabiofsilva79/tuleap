Link selector displays heterogeneous groups of items with different behaviours and lets users select an item among them.

It looks visually like list-picker, but tailored to a different use-case.

Options:

  • templating_callback: (html: HTMLTemplateStringProcessor, item: LinkSelectorItem) => HTMLTemplateResult: A callback function returning a lit/html template result to customize item templates
    It takes two parameters:
    • html: HTMLTemplateStringProcessor The lit/html tag function to help the developer build a secure HTMLTemplateResult.
    • item: LinkSelectorItem The item for which to build the template. See below for its definition.
  • selection_callback: (selected_value: unknown | null) => void: A callback function to be executed each time the user selects a value or clears the selection.
    It takes one parameter: selected_value: unknown | null, which holds the value of the item that was just selected or null when the last selection is cleared.
  • search_field_callback: (query: string) => void: A callback to be executed each time the user types a query in the search input.
    It takes one parameter: query: string: The query typed by the user.
  • placeholder: string: A (translated) text that will be shown when there is no selection in the link-selector. Its purpose is to tell the user what kind of items they will search using this field. For example "Search for an artifact..."
  • search_input_placeholder: string: A (translated) text that will be shown inside the search input. It is shown when the user didn't write anything yet in the search input. Its purpose is to tell the user the kind of input they are supposed to write, so that their search yields results. For example "Id, title..."


Methods:
  • setDropdownContent(groups: GroupsOfItems[]) => void: define the contents of the dropdown by giving it a list of GroupOfItems.
    Each GroupOfItems has these properties:
    • label: string: The label of the group. Think of it as the text content of an <optgroup>.
    • empty_message: string: The translated message that will be shown when there are no items. It is the "empty state" of the group.
    • is_loading: boolean: Show a spinner next to the group's label if true.
    • items: LinkSelectorItem[]: the items of the group.
    • icon: string: an icon to display at the left of the group label.
    Each LinkSelectorItem has these properties:
    • value: unknown: The value that will be passed to selection_callback. It can be any type, usually it's an object.
    • is_disabled: boolean: the item cannot be selected if it's false.
  • resetSelection() => void: reset the current selection.
  • destroy() => void: destroy link-selector and remove all event handlers.
